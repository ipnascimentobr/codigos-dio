Fundamentos de Programação em HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criação e estruturação de páginas na web.

Lista de Tópicos Abordados

  • Tags de cabeçalho: <h1> a <h6>
  • Tags de parágrafo: <p>
  • Marcação de texto: <mark>
  • Texto pequeno: <small>
  • Texto em itálico: <i>
  • Texto sublinhado: <u>
  • Texto forte: <strong>
  • Listas ordenadas: <ol>
  • Listas não ordenadas: <ul>
  • Itens de lista: <li>
  • Links: <a>
  • Régua horizontal: <hr>
  • Texto sobrescrito: <sup>
  • Texto subscrito: <sub>
  • Citação em bloco: <blockquote>

Exemplo de Uso de Tags

Veja um exemplo de como utilizar algumas dessas tags:

  1. <h1>Título Principal</h1>
  2. <p>Este é um parágrafo de exemplo.</p>
  3. <strong>Texto forte</strong>
  4. <ul>
    • <li>Item de lista</li>
    • <li>Outro item de lista</li>
    </ul>
  5. <a href="#">Link</a>

Conclusão

Com estas tags básicas, é possível estruturar conteúdo de maneira eficaz em uma página web.
